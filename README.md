## Deskripsi Singkat
Sistem Pakar Diagnosa Kerusakan Vespa merupakan aplikasi berbasis web yang bertujuan untuk membantu pengguna dalam mendiagnosa kerusakan pada sepeda motor Vespa berdasarkan gejala yang dialami.  
Sistem ini menggunakan metode **Forward Chaining** untuk menentukan jenis kerusakan dan memberikan solusi perbaikan yang sesuai.
